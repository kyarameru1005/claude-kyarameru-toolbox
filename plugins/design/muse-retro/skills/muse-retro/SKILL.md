# muse-retro

70〜80年代風の、懐かしさと温かみのある見た目を与える系統。
`muse-interface`（共通の UI 構造設計）の上に、レトロの見た目トーンを与える。

## この系統の狙い / 与える印象

- 懐かしい・温かい・味がある。記憶を呼び起こす親しみ
- 完璧さより、あえての粗さ・色あせで人間味を出す

## 配色・トーン

- ベースはくすんだ暖色（マスタード・テラコッタ・オリーブ・クリーム）
- 彩度を落とし、色あせたフィルムのような調子に
- 高コントラストは避け、全体をなじませる

## タイポグラフィ

- ヴィンテージ書体（丸みのあるセリフ、太いグロテスク、スクリプト）
- 見出しは大きく、当時の広告のように主張させる
- 字間をやや詰めてレトロな密度感を出す

## レイアウト・余白

- 帯・枠・ストライプなど当時の誌面/パッケージ的な区切り
- 中央寄せやシンメトリーで古さを演出することも
- 余白はほどほど。要素にまとまりを持たせる

## 装飾・質感・モーション

- グレイン（粒状ノイズ）・紙質感・かすれで質感を加える
- バッジ・リボン・サンバースト等のレトロモチーフ
- モーションは控えめ。質感で語り、動きで主張しすぎない

## 避けること（アンチパターン）

- 鮮やかな原色やネオン（テック寄りになり懐かしさが消える）
- フラットで清潔すぎる質感（味が出ない）
- 装飾モチーフの盛りすぎでうるさくなる

## 出力（hephaestus に渡す見た目指針）

- 配色（くすんだ暖色のベースとアクセント）の方向
- ヴィンテージ書体の選び方と見出しの主張度
- グレイン/紙質感の強さと、レトロモチーフの使いどころ
