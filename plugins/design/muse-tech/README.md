# muse-tech

UI の見た目を「テック/サイバー」系統に揃えたいときに使う。暗背景・ネオンアクセント・等幅書体・グリッド線で、配色・タイポ・装飾のトーンを近未来的にまとめる方向付けに向いている。ダッシュボードや開発者向けに。

## 依存

- `muse-interface`

## 動作確認シナリオ

- シナリオ: 開発者向けダッシュボードの見た目を、テック/サイバー系統のトーンに揃えたい。
- 入力例: 「このダッシュボードをテック系のデザインにして。暗背景でネオンっぽい感じで」
- 期待する発火: `muse-tech` スキル。
- 期待する挙動の要点:
  - 暗背景・ネオンアクセント・等幅書体・グリッド線を軸に配色/タイポ/装飾の方向づけを示す。
  - 実装ではなく方針・トーンの提示にとどめる。
- 逸脱の例（NG）:
  - 明るいパステル基調など、別系統（muse-cute 等）のトーンを提示してしまう。
  - 方針提示を飛ばして具体実装コードだけを返す。
